Após download da pasta, é importante renomeá-la para "curvamaster" ao invés de "curvamaster-master" como fica nomeado após extração.
Para compilar o código, vá ate o diretório em que está a pasta com todos os códigos dentro.
Nesse diretório, execute o comando "javac curvamaster/*.java".
Feito isso, dentro da pasta com os códigos serão criados vários arquivos '.class'.
Ainda fora do diretório onde os códigos se encontram, execute o comando "java -cp . curvamaster.CurvaMaster".
Feito isso, o programa deverá ser executado.
